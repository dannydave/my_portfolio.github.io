Heart Disease Diagnosis using Machine Learning and Data Mining

This project was developed as part of my MSc dissertation to predict heart disease using supervised machine learning algorithms. It involved data cleaning, feature selection, and building classification models like Random Forest, Decision Tree, Naive Bayes, and SVM. A simple Tkinter-based GUI was also created for user-friendly predictions. Key skills demonstrated include data preprocessing, machine learning model development, evaluation using performance metrics, and building healthcare-focused AI solutions.
